DOM文档对象模型（Document Object Model）

什么是DOM？

先来甩上来某百科的定义：是W3C组织推荐的处理可扩展置标语言的标准编程接口。它是一种与平台和语言无关的应用程序接口(API),它可以动态地访问程序和脚本,更新其内容、结构和www文档的风格(目前,HTML和XML文档是通过说明部分定义的)。文档可以进一步被处理，处理的结果可以加入到当前的页面。DOM是一种基于树的API文档，它要求在处理过程中整个文档都表示在存储器中。另外一种简单的API是基于事件的SAX，它可以用于处理很大的XML文档，由于大，所以不适合全部放在存储器中处理。

节点

讨论完了什么是DOM，下面来看看节点。节点就是网络中的一个连接点。

根据 W3C 的 HTML DOM 标准，HTML里的所有内容都是节点：

  • 整个文档是一个文档节点
  • 每个 HTML 元素是元素节点
  • HTML 元素内的文本是文本节点
  • 每个 HTML 属性是属性节点
  • 注释是注释节点

简单说说元素节点、文本节点以及属性节点：

元素节点： 本身部包含内容，但是可以包含其他的节点。

文本节点： 一般被包含在元素节点内部，形成了页面文档的主要内容。

属性节点： 用于对元素做出一些具体的描述。

节点操作

获取元素节点

获取元素节点的方式：id选择器、类名选择器、标签名选择器以及 html5 新增的css选择器

id选择器

方式： document.getElementById(“id名”)

getElementById()是document对象特有的函数，传入元素的id名，返回一个对象。这个对象就是传入id对应的节点。

class选择器

方式： document.getElementsByClassName(“class名”)

getElementsByClassName()是传入一个class名，返回一个对象数组。

标签名选择器

方式： document.getElementsByTagName(“标签名”)

getElementsByTagName()是传入标签名(li、div……),返回一个对象数组。

CSS选择器

方式： document.querySelector(“#id”)(选择id名为id的节点)
document.querySelectorAll(“.class”)(选择class名为class的所有节点)

querySelector()返回一个节点，多个匹配之后返回第一个节点。
querySelectorAll()返回节点列表。

获取其他节点

getElementsByName： 通过name属性获取节点。

document.documentElement： 专门获取html标签的。

document.body： 获取body标签。

childNodes： 用来获取一个节点的所有子节点，得到一个包含所有子节点的数组。他会把换行和空格也当成是节点信息，为了不显示不必须的换行的空格，我们如果要使用childNodes就必须进行必要的过滤。通过正则表达式式取掉不必要的信息。

children： 获取子元素，返回数组，按照数组的访问形式即可。

firstChild/lastChild： 获取第一个子元素(最后一个)，和ChildNodes类似，需要过滤。

firstElementChild/lastElementChild： 获取第一个子元素(最后一个)，不用过滤。

parentNode/parentElement： (W3C/ie 标准)获取当前节点的父节点，如果没有父节点返回null。

offsetParent： 获取所有父亲节点

previousSibling/nextSibling： 获取上一个兄弟节点(下一个)，会匹配字符，需要过滤。

previousElementSibling/nextElementSibling： 获取上一个兄弟节点(下一个)，直接匹配节点，不需要过滤。

nodeType： 这个属性是一个数值，让我们可以知道正在处理哪种类型的节点。 元素节点：1；属性节点：2；文本节点：3。

nodeValue： 文本节点的值。

document.write()： 把字符串插入到文档中。

innerHTML()： 读写html的内容。

createElement()： 创建元素节点。参数是标签名。不在DOM树上，无法显示，需要插入。

createTextNode()： 创建文本节点用于放文本内容。不在DOM树上。

appendChild()： 将新创建的节点插入到DOM树，使之成为插入节点的子节点。

removeChild() 删除一个节点，返回被删除的节点。

replaceChild(newNode,oldNode) 替换节点，第一个参数是要插入的节点，第二个参数是被替换的节点。返回被替换的节点。
