简介：

BOM浏览器对象模型（Browser Object Model）

什么是BOM？

这兄弟跟DOM长得好像，还好我会一点英语，看看全称原来是浏览器对象模型，懂了懂了！旁白：你懂啥了你懂。我：我错了大哥，我这就学习！

浏览器对象模型： 这个对象就是对应着浏览器窗口window。它提供了一些方法用于访问浏览器的功能，这些功能和网页内容无关。

window对象

window对象是BOM的核心，是一个全局的变量，表示正在打开的窗口。我们定义的全局变量，本质就是定义在了window上。

1. 全局作用域

全局作用域下定义的变量和函数都是直接归到window的，可以通过window.***来访问到。函数在全局作用域的this是直接指向window的。

2. 系统对话框

alert(“我是警告框”)

confirm(“我是对话框”) 有一个确认和取消按钮，返回对应的布尔值。

prompt(“我是输入提示框”) 确认返回用户输入的文本框的值，取消返回null。

3. location对象

可以访问当前载入的URI（统一资源标识符）的任意信息。

hash： 设置或返回从井号 (#) 开始的 URL(锚)。

host： 设置或返回主机名和当前 URL 的端口号。

hostname： 设置或返回当前 URL 的主机名。

href： 设置或返回完整的 URL。

pathname： 设置或返回当前 URL 的路径部分。

port： 设置或返回当前 URL 的端口号。

protocol： 设置或返回当前 URL 的协议。

search： 设置或返回从问号 (?) 开始的 URL(查询部分)。

4. navigator对象

这个对象提供几个属性，用于辅助检测浏览器环境。详情：https://www.w3school.com.cn/jsref/dom_obj_navigator.asp

5. history对象

通过用户访问产生的浏览历史来向前或向后移动的方法。
