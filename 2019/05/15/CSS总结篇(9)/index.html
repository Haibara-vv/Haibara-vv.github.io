CSS总结篇(9)

发表于 2019-05-15

简介：
    经过这么长时间的前端学习，现在也在js和jq的学习了，但在最近的项目里常常很多的CSS样式不记得了，于是打算花一晚上总结一下常会用到的CSS样式。虽然W3C School已经都有了，但是想着自己总结一遍印象会更深刻。如下：
ps: 篇幅有限，本篇总结了CSS的一些核心属性(再续上篇)，本篇包括：背景相关属性、弹性盒子，动画等。

背景相关属性：

background-clip:    背景图展示位置    图片必须平铺
    border-box    边框区域开始展示
    padding-box    内边距区域开始展示
    content-box    内容区域开始展示

background-origin:    背景图起始位置
    border-box    边框区域开始
    padding-box    内边距区域开始
    content-box    内容区域开始

background-size:背景图大小;
    具体px
        一个值    指定宽度，高度等比例缩放
        两个值    指定宽高

%
cover 等比例缩放图片，且充满整个元素，有可能图片超出元素
contain    等比例缩放图片，首先充满最近的一边，立即停下，图片有可能未充满整个元素

border-image:url("图片路径") 图片裁剪的大小 图片的铺放方式;图片边框
border-image-repeat:铺放方式;
    repeat
    round
    stretch

border-image-slice:图片裁剪;
border-image-source:url("图片路径");
border-image-width:图片边框的宽度;

颜色：
    hsl(色调,饱和度%,亮度%)
        0-360
        0-1
        0-1

    hsla(色调,饱和度%,亮度%,透明度)
        0-360
        0-1
        0-1
        0-1

弹性盒子

标准盒模型 :  box-sizing:content-box;
怪异盒模型 :  box-sizing:border-box;
    W = w(padding+border) + marging
    H = h(padding+border) + marging

触发方式：
丢失<!DOCTYPE html> 在ie6 7 8 下触发
box-sizing:border-box;

弹性盒模型

主轴 侧轴（交叉轴）        
容器 （父元素）
    主轴
    侧轴
display:flex/inline-flex;
        特性：子元素的浮动，清除浮动，垂直对齐方式都失效
            默认水平方向主轴
            项目的宽高默认由内容撑起来
            项目的高度默认与容器的高度一致
            项目默认按照主轴排列
flex-direction:column/column-reverse/row/row-reverse;

垂直方向/垂直方向且反序/水平方向默认值/水平方向且反序

flex-wrap:nowrap/wrap/wrap-reverse;项目是否换行
不换行/换行/反序换行

flex-flow:主轴方向 换行;

justify-content:center/flex-start/flex-end/space-around/space-between;
    居中/起始位置/终点位置/平均分配/两端对齐
align-items:center;侧轴上居中
align-items:flex-start;侧轴上的起始位置
align-items:flex-end;侧轴上的终点位置
align-items:stretch;默认值
align-items:baseline;第一行文本底部对齐

align-content:center;多根轴线在侧轴上的对齐方式
    flex-start    起始位置
    flex-end    终点位置
    center        居中
    stretch        默认值
    space-around    平均分配
    space-between   两端对齐
项目 （子元素）
order:数值;    定义项目排列顺序
    默认值为0，数值越大越靠后，数值越小越靠前
    flex-grow:;项目放大比例，将剩余空间分成等份分给大家
    flex-shrink:项目缩小比例; 默认值1
    flex-basis:; = width 项目本身的大小
        auto 默认值
    flex:;三个值的综合写法

    align-self:项目在侧轴上的对齐方式;
        flex-start    起始位置
        flex-end    终点位置
        center        居中
        auto        自适应
        baseline    第一行文本底部对齐
        stretch    默认值

移动端布局

加meta属性
<meta name="viewport" content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no" />

媒体查询
响应式布局
流体网格
分栏布局

column-count:;分栏数
column-width:;多宽为一列
column-gap:;栏之间的间距
column-rule:1px solid red;栏线
column-span:all/1(默认);分栏标题

背景渐变
默认从上到下的写法

background:linear-gradient(bule,red);
background:-webkit-linear-gradient(blue,red);

设置了渐变角度的写法

background:-webkit-linear-gradient(right top,red,yellow);
background:linear-gradient(to left top,blue,red);

带有角度的写法

background:linear-gradient(90deg,red,blue);
background:-webkit-linear-gradient(90deg,red,yellow);

background:linear-gradient(blue 20%,red 50%);
background:-webkit-linear-gradient(blue 20%,red 50%);

重复渐变

background:repeating-linear-gradient(#000 10%,#fff 20%);
background:-webkit-linear-gradient(red 15%);

background:radial-gradient(orange,red);
background:-webkit-radial-gradient(orange,bkue,yellow,red,green,#000);

background:radial-gradient(red 15%,yellow 30%,blue 55%);
background:-webkit-radial-gradient(red 15%,yellow 30%);

设置了圆的形状 正圆 椭圆

background:-webkit-radial-gradient(circle,orange,blue,yellow,red,green,#000);
background:radial-gradient(circle,orange,blue,yellow,red,green,#000);

background:repeating-radial-gradient(red 5%,orange 10%,blue 15%,pink 20%);
background:-webkit-radial-gradient(#000 5%,blue 10%)

模糊

filter:blur(10px);
filter: progid:DXImageTransform.Microsoft.Blur(PixelRadius=10,MakeShadow=false); 兼容ie6 7 8 9

倒影

-webkit-box-reflect:above 10px linear-gradient(transparent,#fff);
-box-reflect:倒影的方向 原图与倒影的距离 倒影是否渐变;
above 向上
below 向下
left  向左
right 向右
注意：倒影有渐变，必须有间距

渐变
transition-property:width; 过渡属性
transition-duration:2s; 过渡时间
transition-timing-function:ease; 运动方式
transition-delay:3s; 延迟时间
transition:;

动画

先定义动画

@keyframes 动画名{
    from{}
    to{}
}

再调用动画

animation:动画名 时长 延迟 运动方式 重复次数 是否回播;
animation-name:name,name2;
animation-duration:2s;
animation-delay
animation-timing-function:linear;
animation-iteration-count:infinite;
animatio-direction:alternate;

转换 transform

transform:translate();
    translateX()
    translateY()

transform:rotate(90deg);

transform:scale(1.5);    放大缩小
    允许负值    先缩小到消失再出现
    绝对值大于1：放大
    绝对值等于1：不变
    绝对值小于1：缩小

transform:skew(50deg,50deg);倾斜

transform-origin:;改变元素基点

3D环境

transform-style:preserve-3d;提供3D环境 给所有变形元素的父级添加
perspective:500px;景深
perspective-origin: 100px 50px;
