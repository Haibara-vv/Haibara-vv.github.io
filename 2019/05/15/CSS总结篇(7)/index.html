CSS总结篇(7)

发表于 2019-05-15

简介：
    经过这么长时间的前端学习，现在也在js和jq的学习了，但在最近的项目里常常很多的CSS样式不记得了，于是打算花一晚上总结一下常会用到的CSS样式。虽然W3C School已经都有了，但是想着自己总结一遍印象会更深刻。如下：
ps: 篇幅有限，本篇总结了CSS的一些核心属性(再续上篇)，本篇包括:CSS文档统筹、h5历史等。

css文档统筹

代码部分
    <meta name="Keywords" content="关键词"/>
    <meta name="Description" content="描述/简介"/>

页面体积尽可能小，大面积使用的代码，提取出来

页面优化：
    尽可能使用文本链接，少使用img链接或者flash，或者视频
    合理使用友情链接
    合理的代码结构
    命名规范
        小写的英文字符，不能数字开头，见名知意，简写

    css重置的目的:方便程序员更好的控制页面的样式设置

h5历史：

盒模型的分类：
    标准盒模型：
    w = width + 左右内外边距 + 左右边框
    h = height + 上下内外边距 + 上下边框

怪异盒模型：
    w = width(左右内边距，左右边框)    + 左右外边距
    h = height（上下内边距，上下边框）+ 上下外边距


触发怪异盒模型：
    丢失<!doctype html>    在ie6 7 8下触发
    box-sizing:;
    border-box    怪异盒模型
    content-box    标准盒模型

语义化标签：

section        内容区域
header        头部区域
footer        底部区域
article        与上下文不相关的独立内容区域
aside        与article相关的独立辅助信息
nav            类似无序列表的
figure > figcaption:    一段独立的流内容，只有一个figcaption
embed        定义嵌套内容    媒体标签
mark        高亮标签

video 视频标签

source         备份文件，引入各种格式的视频文件
    ogg
    wenm
    mpeg4
controls    显示控件
poster        第一帧显示的图片
autoplay    自动播放（控件不显示）
    preload        预加载，预播放
    loop        循环播放
audio        音频标签

表单元素：

input    表单元素
    multiple="多个邮箱地址"
    min="最小值"
    max="最大值"
    step="步长"

    type="email"    邮箱地址
    type="url"        网址
    type="range"    左右滑块
    type="number"    数字输入框
    type="date"        日期
    type="month"    月份
    type="week"        周期
    type="time"        时间
    type="datetime"    utc时间（大部分浏览器不支持）
    type="datetime-local"    本地时间


    autofocus="默认光标位置"
    required="不能为空"
    autocomplay="on/off/空"    记录历史记录
        注意：被记录的表单元素必须有name值
    novalidate="取消验证"
    maxlength="长度"
    placeholder="提示信息"

datalist > option    自定义下拉列表
    id="name" 与 input 中 list="name"

    input    type="range"    滑块
        typ="number"    数字输入框
        output            输出框  对标签
            for="关联相关的标签id的值"
    hgroup        标题分组标签
