CSS总结篇(4)

发表于 2019-05-15

简介：
    经过这么长时间的前端学习，现在也在js和jq的学习了，但在最近的项目里常常很多的CSS样式不记得了，于是打算花一晚上总结一下常会用到的CSS样式。虽然W3C School已经都有了，但是想着自己总结一遍印象会更深刻。如下：
ps: 篇幅有限，本篇总结了CSS的一些核心属性(续上篇)

盒模型 

内容区域        content
内边距区域    padding
边框区域    border
外边距区域    margin

padding-top:顶部内边距;
padding-right:右内边距;
padding-bottom:底部内边距;
padding-left:左内边距;
padding:四个方向;
padding:上下 左右;
padding:上 左右 下;
padding:上 右 下 左;
注意：没有负值。没有auto，会撑大元素的大小，则需要减少相应的宽高

margin-top:顶部外边距;
margin-right:右外边距;
margin-:bottom:底部外边距;
margin-left:左外边距;
margin:四个方向;
margin:上下 左右;
margin:上 左右 下;
margin:上 右 下 左;

文本溢出

overflow:文本溢出;
overflow-x:水平文本溢出;
overflow-y:垂直文本溢出;
    visible 默认值
    hidden 超出隐藏
    scroll 出现滚动机制
    auto   自适应
    inherit 继承

空白区域 

white-space:空白区域;
    normal    默认值，空白会被浏览器忽略，
        ⭐nowrap    文本不会换行，文本会在同一行上继续，直到遇到<br/>标签为止；
    pre    空白会被浏览器保留，其行为方式类似HTML中的pre标签；
    pre-wrap    保留空白符序列，不换行；
    pre-line    合并空白符序列，但是保留换行符；
        inherit    规定应该从父元素继承White-space属性的值；(ie浏览器不支持此属性值)

省略号 

text-overflow:省略号;
clip    裁剪
ellipsis    省略号

显示省略号，只针对单行文本
overflow:hidden;
white-spacing:nowrap;
text-overflow:ellipsis;

元素类型

块级元素 内联元素（行内元素） 内联块状元素（行内块元素） 可变元素

块级元素 block
    单独占据一行
    可以设置宽高
    内外边距能够正常设置
    上下排列
    一般用来搭建页面结构
    div h1-h6 ol ul li dl dt dd p……

内联元素 inline
    只占据自己内容位置大小
    不能设置宽高
    垂直方向的内外边距不能正常设置
    左右排列
    span i b em strong a……

内联块状元素 inline-block
    只占据自己内容位置大小
    能设置宽高

元素类型的转换

display:;        18个属性值
block / inline / inline-block / none / list-item 转换为li默认样式
注意：float之后的元素具有块级元素特性

元素垂直对齐方式 

vertical-align:;
⭐top / middle / bottom / baseline / text-top 文字顶部对齐
/ text-bottom 元素与文字底部对齐

可以消除img下方自带的3px的间距

定位

定位的作用：改变元素位置
    改变元素坐标的属性: top bottom left right
position:;

static    默认值 静止
absolute 绝对定位
    脱离文档流
    参考最近的有定位的父级元素，如果没有，则参考根元素(html)

relative 相对定位
    占据文档流
    参考自身默认加载出来的位置

fixed    固定定位
    脱离文档流

层级关系：
z-index:改变层级关系;
    auto 默认值
    number 没有单位的整数值越大，层越高,值越小，层越低

并列关系(兄弟关系):
添加 了position:relative / absolute /fixed;的元素能修改层级关系

嵌套关系(父子关系):
元素添加了position:relative / absolute; 只能改动子元素的z-index:;
如果父元素添加了position:fixed;元素将不能改动z-index:;
如果只有子元素添加了position:fixed; 子元素可以改动z-index:;

包含块：
嵌套关系，且父级有定位是position:relative;子元素有定位position:absolute;
