CSS总结篇

发表于 2019-05-15

简介：
    经过这么长时间的前端学习，现在也在js和jq的学习了，但在最近的项目里常常很多的CSS样式不记得了，于是打算花一晚上总结一下常会用到的CSS样式。虽然W3C School已经都有了，但是想着自己总结一遍印象会更深刻。如下：

标签的基本语法：
单标签 空标签
<标记 属性=”值” 属性=”值”/>
对标签 双标签
<标记 属性=”值” 属性=”值”> </>
基本标签：
h1-h6：标题标签
特性:单独占一行，文本加粗变大，依次变小，
h1在一个页面中有且只有一个，其他的可以多次出现

p:段落标签 
    特性：独占一格，表示一个段落，不能自己嵌套自己，不能嵌套块级元素
    一般嵌套 span a img 
&nbsp:特殊符号 更多：https://www.cnblogs.com/knowledgesea/p/3210703.html
br:换行标签
b / strong: 文本加粗标签；
        特性：只占据自己内容大小的位置，strong有利于优化搜索引擎
i / em: 文本倾斜标签
        特性：只占据自己内容大小的位置，文本倾斜，em有利于搜索引擎优化    
hr: 水平线标签 空标签 块标签
    特性：占据一行，双边框线
ul > li :无序列表
        特点 ：单独占一行，小黑圆点。 list-style:none去圆点
        type="disc/circle/square" 修改样式 实心圆/空心圆/方块
        一般用来做单导航    
ol > li :有序列表
        特点 ：单独占一行，默认使用阿拉伯。 list-style:none
        type="1/a/A/i/I" 修改样式  strat=""从哪一个开始排序
        一般用来做单导航    
dl > dt + dd : 自定义列表
        特性：单独占据一行，一个dl中只有一个dt，多个dd。dd是对dt的解释。
        一般用来做双导航    
img : 图片标签
    .jpg 常用的图片
    .png 背景透明的图片
    .gif 动态图 帧
    属性：
        src="图片路径" 
            绝对路径
                从盘符出发
                    src="C:\Users\SSW\Desktop\point.png"
                域名地址(路径) http://
            相对路径
                相对指定的文件夹引入的图片
                src="img/2.jpg"
                src="../2.jpg"    
                ./ 代表文件所在目录
                ../ 代表文件所在父级目录
                ../../ 代表文件所在的父级目录的父级目录
                / 代表文件所在的根目录        
        alt="文本提示"
            图片加载失败时显示
            提供搜索信息
        title="鼠标悬停的文本提示"()
        width="宽度"
        height="高度"
            只写宽/高，另一边等比缩放。
        特性：
            插入一张图片，只占据自己内容大小的位置，能够改变大小                
a: 超链接。实现页面与页面之间的跳转
    href="跳转网址" # 空链接
    title="鼠标悬停时的文字"
    target="_blank" 页面打开的方式
            _self 当前窗口刷新   _blank 不断打开新窗口  _new新窗口中刷新
特性： 文本蓝色，，带有下划线，只占据自己内容的大小的位置，鼠标显示小手    
div： 盒子标签，块级元素
    特性：单独占据一行，没有特殊的视觉效果，一般用来搭建页面的整体框架        
span： 文本节点标签
    特性：只占据自己内容大小的位置，没有特殊的视觉效果，一般用来结合css显示指定的样式            
table > tr >td:   表格标签 行 单元格
            border="" 边框
            bordercolor="" 边框颜色
            bgcolor=""背景颜色
            只能给table添加
            cellspacing="" 边框与边框之间的间距
            cellpadding="" 边框与内容之间的边距                
            colspan="" 合并列
            rowspan="" 合并行
            align="" 水平对齐方式
                center/left/right    
form: 表单  
    作用：收集用户信息
     action="提交到哪里去"
     method="提交方式" get/post
     name="表单名称"         
     表单元素：
     特性：只占据自己内容的大小
        input
            name="表单元素的名字"
            placeholder="提示信息" css3 特有的
            value="内容"
            type=""
                text    单行文本输入框
                password    密码输入框
                radio    单选框
                    name="值"    同一组选项的name值必须一致，必须有name属性
                    checked="checked"     默认选中项
                    disabled="disabled"    禁用
                checkbox    多选框
                    checked="checked"     默认选中项
                    disabled="disabled"    禁用
                submit    提交按钮
                reset    重置按钮
                button    普通按钮，没有功能，需要结合其他代码赋予功能
                <button></button>    提交标签，有提交功能，对标签                
        select > option    下拉列表                    
        textarea：    多行文本输入框
            cols="字符宽度"
            rows="行数"
            默认可以拖拽大小，禁止拖拽resize:none;    
            当内容超出多行文本区域，则出现滚动条预览

标记的语法：
<标记 属性=”值”></标记>
<标记 属性=”值” />
标签是写在body标签中间
text-decroation: ;文本修饰
