CSS总结篇(5)

发表于 2019-05-15

简介：
    经过这么长时间的前端学习，现在也在js和jq的学习了，但在最近的项目里常常很多的CSS样式不记得了，于是打算花一晚上总结一下常会用到的CSS样式。虽然W3C School已经都有了，但是想着自己总结一遍印象会更深刻。如下：
ps: 篇幅有限，本篇总结了CSS的一些核心属性(再续上篇)

锚点链接的作用：
实现当前页面的不用位置的跳转

点击位置        
<a href="#name"></a>
跳转位置        
<标记 id="name"></标记>
<a name=""></a>

透明：

opacity:数值;
filter:alpha(opacity=数值);  0-100

flash动画:

<object>
    <param name="movie" value="文件地址" />
    <embed src="文件地址" wmode="transparent" />
</object>

内联框架:

<iframe src="网址" />
    frameborder="0" 0没有边框 1有边框
    width=""
    height=""
    scrolling="yes" yes可以滚动 no不能滚动

滚动标签:

<marquee>

图片整合: 优点：减小图片体积；减少服务器访问次数

自适应：

width: %;
height:auto/不写;
最小宽高 /最大宽高
    min-width:;
    min-height:;

过滤器：

下划线属性过滤器
_height:;ie6识别

显示隐藏： 

visibility:可见/不可见;
visible 可见
hidden 不可见
visi

页面中的宽高自适应：

宽度自适应：    width:100%;
高度自适应：    height:auto/不写;

min-width:最小宽度;
min-height:最小高度;
max-width:最大宽度;
max-height:最大高度;
注意：    ie低版本不识别

过滤器：

_height:;    兼容ie6-
height:100px !important;    不兼容ie6-

+height:;    兼容ie低版本
*height:;    兼容ie低版本

height:100px\0;    兼容ie8
height:100px\9;兼容ie8

:root .box{}    不兼容ie8-，其他浏览器都识别

visibility:显示隐藏;

hidden    隐藏
visible    显示
visibility与display区别：
visibility:hidden;    占据文本流
display:脱离文本流;

伪类

.box:after{content:"";}
在box元素的内容之后添加伪对象，此对象跟普通标签一样

.box:before{content:"";}
在box元素的内容之前添加伪对象，此对象跟普通标签一样

万能清除浮动法:
.clearfix:after{
    content:"";
    display:block;
    width:0;
    height:0;
    overflow:hidden;
    visibility:hidden;
    clear:both;
}


:first-letter{}    选中第一个字
:first-line{}    选中第一行
:selection{}    文本被选中时的文字颜色和背景颜色
