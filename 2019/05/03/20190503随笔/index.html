简介：
    趁放假去同学们实训的培训地点玩儿，蹭一节实训课，随便做了点笔记，本笔记是关于HTML的一个概述，碎片化的笔记，以后回顾扩展的时候用吧。

H5

html5—–html4转变而来(更新？)
html： 超文本标记语言
用途： 办公 随着web的发展 html被广泛使用
html 是用于构建的网页的原材料
html5 相对于html4的改变，对于移动端更加友好 对于移动端的适应
html5包含的内容有：
html 超文本标记语言
css 层叠式样式表
js javascript脚本 vue.js node.js跟数据库可以交互 react….
h5可以干什么?
pc端的网页 京东淘宝…. 移动端的网页… 小程序…微信公众号…app

html 当做文档 自己的特殊的地方 标记 独立的一门语言
文档 布局
做网页的如何跟浏览器交互使用
任何去使用标记
行内标签 在一行内不会换行的 没有宽高
span b i u em strong del a td …
块标签 默认占一行会换行的 有宽高
div p h1-h6 ul>li ol>li tr form..
行内块标签 在一行内不会换行但是有一定的大小（有宽高）
img input..
标记 就是在网页的不显示的东西 尖括号括起来 有开始有结束双标记 也有只有一个的单标记
语言 语法 没有语法 全靠记
结构 

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8"/>
    <title></title>
  </head>
  <body></body>
</html>

CSS 层叠式样式表 装饰工 在html中写要放在style标签中 style标签放在title后面的head里面 外链式 link导入
选择器
id选择器 id是唯一的 #id名 class选择器 .class名 可以选择多个 标签选择器 选择多个 标签名
伪类选择器 :first-child 选择第一个 :last-child是选择最后一个
选择器优先级
id 100 > class 10 > 标签选择器 1 > 伪类选择器 0.8 他会把所有的选择器加到一块
基本样式
字体颜色 color 字体的设置 font开头 font-size…
文本对齐方式 text-algin （center,left,right）
背景设置 background开头的 background-color，background-size 背景大小设置…
盒子模型 内填充 padding 外边距margin 外边框border（border和padding都会改变盒子的大小）以px像素为单位的
padding:10px; ===> padding：10px 10px 10px 10px;(顺序上右下左) ===>padding:10px 10px;(上下的填充,左右的填充)
宽，高 （仅针对于存在宽高的）width height
行间距设置 行高设置 line-height （行高和高设置为一样的时候可以实现文字的垂直居中）
边距的使用 margin:0 auto; 自动去块的居中对齐 基于盒子宽度已知
边框的使用 border:1px solid red; 边框的宽度 边框的线的样式 边框的颜色
定位 可以实现把标签放在网页的任意位置 position
相对定位 relative占用原来位置 相对于原来的位置来定位
绝对定位 absolute不占用位置 相对于浏览器来定位的 如果他的父类设置了相对定位那么就相对于 父类来定位（子绝父相）
固定定位 fixed不占用位置 相对于浏览器来定位 不会随着滚动栏滚动
块和行内元素的转换 行内元素转为块元素 display:block; 行内元素转为行内块元素 display:inline-block;
浮动 把俩个标签放到一行 把俩个东西浮到水面他就在同一行了 一般设置为float:left;
溢出隐藏 overflow:hidden;(清除浮动)
CSS3新特性
transform 2d转换 在水平面和垂直面的转换
translate 平移 x轴 从左往右 往左移为负值 往右移是正值 Y轴从上到下 往上移是负值 往下移正值 px
rotate 旋转 单位 deg度数 旋转从上往下转 沿x轴 从左往右 沿y轴
scale 缩放 大于1就是放大 小于1就是缩小
skew 偏移 单位deg度数
border-radius 圆角边框 左上角的 右上角 右下角 左下角 单位可以写像素也可以写百分比
border-radius: 10px 10px 10px 10px;
transition 缓动 给你添加一个过渡的动画 需要添加动画样式 动画的执行时间 all全部所有的都添加
transition:all 0.5s;
阴影
文本阴影 text-shadow 盒子阴影 box-shadow
rgba颜色 r红色 最大为255 g是绿色 b蓝色 a透明度 最大为1（1表示不透明）
动画
定义动画
@keyfarms 动画名{
form{开始}
to{结束}
}
@keyfarms 动画名{
百分比来控制
0%{初始的时候}
50%{一半的时候}
100{结束的时候}
}
调用动画 放在调用者的样式中 匀速 linear 循环 infinite
animation: 动画名 动画执行一次的时间 动画的运动轨迹 动画的执行完的状态
opcity 透明度设置
